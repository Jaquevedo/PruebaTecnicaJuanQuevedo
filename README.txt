# Aplicación Spring Boot - Prueba Técnica

## Descripción
Esta aplicación ha sido desarrollada utilizando Spring Boot como parte de una prueba técnica. El desarrollador de la misma es **Juan Quevedo**.

## Requisitos Previos
Asegúrese de tener los siguientes elementos instalados y configurados:
- Java 17 o superior.
- Maven 3.8.1 o superior.

## Documentación
La documentación de la API está disponible en el siguiente enlace cuando la aplicación esté en ejecución:

[Documentación de la API](http://localhost:8080/ApiDoc)
*http://localhost:8080/ApiDoc*

## Características principales
- **Arquitectura REST**: Estructura escalable y modular basada en principios RESTful.
- **Swagger**: Interfaz para explorar y probar los endpoints de la API.

## Tecnologías utilizadas
- **Spring Boot**: Framework principal del proyecto.
- **Maven**: Gestión de dependencias y construcción del proyecto.
